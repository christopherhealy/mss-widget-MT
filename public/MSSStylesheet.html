MSS UI Style Guide

Buttons & controls currently used in Widget, Report, and Admin tools.

v0.1 · audit only

Buttons

Primary action button
Class: .mss-btn.primary / .btn.primary
Used in: Widget.html (“Record your response”, “Submit for scoring”), report.html (“Refresh”)
Props: filled, accent background, soft shadow, pill radius.
<button class="mss-btn primary"> Primary action </button>
Secondary / default button
Class: .mss-btn / .btn
Used in: Widget “Stop”, “Clear file”, “Close”, Admin and Report secondary actions.
Props: neutral gray background, pill radius, hover darkens.
<button class="mss-btn"> Secondary button </button>
Outline button
Class: .btn.outline
Used in: report.html (“Download CSV”).
Props: white background, 1px border, subtle hover.
<button class="btn outline"> Outline button </button>
Navigation pill
Class: .mss-pill
Used in: Widget.html (“Previous”, “Next” question).
Props: bordered pill, white background, hover gray.
<button class="mss-pill">Previous</button> <button class="mss-pill">Next</button>

Inputs & Filters

Search / filter input
Classes: .mss-input (proposed shared), or plain <input type="text"> styled in Report/Admin.
Used in: report.html filter box (“student, question, CEFR…”).
<input type="text" class="mss-input" placeholder="Filter by student, question, CEFR…" />
Limit select
Control: <select>
Used in: report.html (“Last 100 / 50 / 25…”).
Styled with the same pill geometry as inputs.
<select> <option>Last 25</option> <option>Last 50</option> <option>Last 100</option> </select>

Chips & Tags

Score / CEFR chip
Class: .chip
Used in: report.html for CEFR / score badges and filters.
Props: small pill, subtle border, accent dot.
CEFR C2 TOEFL 29
<span class="chip"> <span class="chip-dot"></span> CEFR C2 </span>
